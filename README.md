# Yogesh M S - Portfolio

A modern, responsive developer portfolio showcasing my projects, skills, and professional experience as a Full-stack Developer and DevOps Engineer.

🌐 **Live Site**: [https://yogesh.trojanx.in](https://yogesh.trojanx.in)

## ✨ Features

- **Modern Design**: Clean, professional interface with smooth animations and transitions
- **Responsive**: Fully responsive design that works on all devices
- **Dark Theme**: Eye-friendly dark theme with vibrant accent colors
- **Interactive Projects**: Showcase of real-world projects with live demos and GitHub links
- **Skills Visualization**: Interactive tech stack display with categorized skills
- **SEO Optimized**: Comprehensive meta tags for better discoverability
- **Fast Loading**: Optimized performance with Vite

## 🛠️ Tech Stack

### Frontend
- **React 18** - Modern UI library
- **Vite** - Next-generation frontend build tool
- **TailwindCSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Router v7** - Client-side routing

### UI Components
- **Radix UI** - Accessible component primitives
- **Lucide React** - Beautiful icon set
- **React Icons** - Popular icon libraries
- **Prism.js** - Syntax highlighting

### Features
- **Lenis** - Smooth scroll
- **Icon Cloud** - 3D interactive icon cloud
- Custom animations and effects

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/lcrank/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 📦 Build for Production

```bash
npm run build
```

The optimized files will be in the `dist` folder.

## 🌐 Deployment

This portfolio can be deployed to various platforms:

### Vercel (Recommended)
```bash
npm install -g vercel
vercel
```

### Netlify
1. Connect your GitHub repository
2. Build command: `npm run build`
3. Publish directory: `dist`

### GitHub Pages
```bash
npm run build
# Deploy the dist folder to GitHub Pages
```

## 🎨 Customization

To customize this portfolio for your own use:

1. Update personal information in `src/pages/Hero/Hero.jsx`
2. Replace projects in `src/pages/Projects/testProjects.jsx`
3. Update skills in `src/pages/Skills/Skills.jsx`
4. Modify experience in `src/pages/Experience/Experience.jsx`
5. Update meta tags in `index.html`
6. Replace images in `src/assets/images/`
